## What Is an Agent Skin?

A **skin** re-skins an agent desktop app's UI without modifying signed application files. Most of these apps are Electron/Chromium-based, so the common approach is **CDP (Chrome DevTools Protocol) injection** — binding to localhost and layering custom CSS/JS/images over the live UI. Native controls keep working; changes are reversible; your API keys and config are never touched.

Two layers people skin:

- **Visual skin** — colors, wallpapers, bubbles, icons, animations, character art.
- **Personality skin** — system prompt + tone that give the same model a different persona *(out of scope for this list, which focuses on visual skins & theming tools)*.
